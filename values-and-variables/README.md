# Primitive Types

- **Number**
- **String**
- **Boolean**
- **Null**
- **Undefined**
- Technically there are two others: **Symbol** and **BigInt**

# Variables

Variables are like "labeled jars" for a value in JavaScript. We cab store a value and give it a name, so that we can...

- recall it
- use it
- or change it later on.
